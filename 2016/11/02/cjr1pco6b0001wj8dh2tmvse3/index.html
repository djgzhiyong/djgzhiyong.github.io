1.创建django项目

在命令行下输入django-admin startproject myproject （myproject即为创建的项目名称）
此时无任何消息输入即成功创建django项目
进入创建成功的项目 myproject 目录下
输入 python manage.py startapp firstapp 创建django应用firstapp

2.使用 Atom 来进行开发

Atom 是 github 推出的一款完全免费的编辑器，各种丰富的插件拓展，推荐使用。
下载地址 https://atom.io/

使用 atom 打开创建的 myproject 项目文件夹

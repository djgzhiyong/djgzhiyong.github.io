发表于 2016-12-19   |  

基于 python 2.7 和 django 1.10

前面5个章节的简述，我们对 django 的基本用法有了初步的了解，从现在开始我们尝试使用 django 编写一个 blog 平台，来更深入的了解 django 的强大。为了使我们的第一个项目有点那么回事，页面不会那么low，我们需要用 bootstrap 来美化下我们的页面，可能需要修改下基本的 CSS 样式，使他更符合我们的意思。还会使用 javascript 来做基本的表单验证，以避免不必要的表单提交。下面就开始…

我们继续在前面章节的基础上编写我们的 blog 应用，在命令行下进入我们的 myproject 项目，输入命令 python manage.py startapp blog 来创建 blog 应用，完成后用 atom 打开 myproject 项目，在项目的 settings.py 文件中安装我们新建的 blog 应用，完成后如下
